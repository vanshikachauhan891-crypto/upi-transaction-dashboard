# 💳 UPI Transaction Dashboard

## 📌 Overview
This project is a Power BI dashboard that analyzes UPI transaction data to provide insights into financial performance. It helps track transactions, revenue, payment trends, and fraud rate.

---

## 🎯 Objectives
- Analyze UPI transaction patterns  
- Monitor revenue and performance  
- Identify fraud transactions  
- Provide business insights  

---

## 📊 Features
- Total Transactions & Revenue tracking  
- Average Transaction Value analysis  
- Payment method distribution (UPI, Card, Cash)  
- Fraud Rate monitoring  
- Interactive dashboard  

---

## 📈 Dashboard Insights
- Shows transaction trends over time  
- Identifies top-performing categories  
- Highlights customer payment behavior  
- Detects high-risk transactions  

---

## 🛠️ Tools Used
- Power BI  
- Excel / Dataset
